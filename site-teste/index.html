Teste de comentário no commit

Testando comando de comentário do commit, com título e descrição do commit, utilizando as instruções do site:

OBS: Procedimento comigo funcionou no terminal gitbash, onde consegui pular a linha com o enter sem enviar o commit antes de fechar as aspas. No terminal prompt de comando ao dar enter ele já envia o commit ao invés de pular a linha.

O :memo: é um emoji, aparecerá no repositório remoto no Título. 


// Adicione
$ git add .

// Faça o commit
$ git commit -m ":memo: Adiciona README
>
> Foi criado o arquivo README.md com as informações mais relevantes do repositório"
// Para realizar a quebra de linha pelo terminal, você terá que abrir aspas " e clicar enter sem fechar as aspas " 
// Após finalizar o título e o corpo do commit, você pode fechar as aspas "

// Por fim, faça o push: 
$ git push origin master
